# modular-synth
Entwicklung eines webbasierten modularen Synthesizers mit integrierter Modulentwicklung.
Die Online-Version gibt es [hier](https://jan-kolpakov.de/).

## Install
```js
npm install
```

## Run
```js
npm run watch
```

## Deploy to jan-kolpakov.de
```js
npm run deploy -- --pw=[Password]
```
